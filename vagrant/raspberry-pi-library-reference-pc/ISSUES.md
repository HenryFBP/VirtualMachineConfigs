## PCmanFM

- Disable PCManFM 
- and stop CTRL-ALT-F<n> keys from changing TTY
- Disable Chrome `...` menu - it opens pcmanfm
- Disable CTRL-ALT-T
- Disable Chrome "Show in Folder"
- Sudo needs to be disabled or have a password
